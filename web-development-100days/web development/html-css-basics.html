HTML & CSS BASICS Summary

HTML Summary

HTML is the heart of our webpages. Without HTML our pages would just be text. It is used for structing the pages, while CSS is used for the decorating.

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag content and then a closing tag.

  • You can also have void (empty) elements like images
  • You can also confgure elements with attributes
  • There's a long list of avaiable elementsbut you'll gain expereince over time, no worries.

Learn more about all avaiable HTML elements on the MDN HTML element reference.
